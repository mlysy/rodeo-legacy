Covariance Function¶

BayesODE.cov_fun.cov_prior(tseq, model, gamma, alpha)¶

Calculate covariance and cross-covariance matrices between :math:x_t and :math:v_t.

Parameters

  • tseq (float) – Vector of N timepoints at which to calculate the covariances.

  • model (str) –

    Name of the covariance model to use. Possible choices are:

    • exp2: Squared-exponential.

    • rect: Rectangular.

    • exp: Exponential.

  • gamma (float) – Covariance decorrelation time.

  • alpha (float) – Covariance scale parameter.

Returns

Three N x N matrices:

  • Sigma_vv = cov(v(tseq), v(tseq))

  • Sigma_xx = cov(x(tseq), x(tseq))

  • Sigma_xv = cov(x(tseq), v(tseq))

Return type

numpy array

ODE Bayes Solver¶

BayesODE.ode_bayes.ode_bayes(fun, tseq, x0, Sigma_vv, Sigma_xx, Sigma_xv)¶

Bayesian solver of ODE problem \(dx_t/dt = f(x_t, t)\).

Parameters

  • fun (function) – ODE function, taking two float parameters and returning a float.

  • tseq (float) – Vector of N timepoints at which \(x_t\) will be calculated.

  • x0 (float) – Initial condition \(x(t_0) = x_0\).

  • Sigma_vv (float) – N x N prior covariance matrix cov(v(tseq), v(tseq)).

  • Sigma_xx (float) – N x N prior covariance matrix cov(x(tseq), x(tseq)).

  • Sigma_xv (float) – N x N prior cross-covariance matrix cov(x(tseq), v(tseq)), where \(v_t = dx_t/dt\).

Returns

A vector of length N corresponding to the probabilistic solution x(tseq).

Return type

float
